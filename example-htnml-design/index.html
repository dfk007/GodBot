1 Chat Cannot Trigger Orchestration Workflows RESOLVED

Chat uses ToolExecutor for CRUD only. Multi-step workflows (e.g. "Find 5 backend jobs in SF and apply") do NOT invoke the LangGraph orchestrator.

Current
  • User ‚Üí Chat Router ‚Üí ToolExecutor ‚Üí list_applications ‚Üí Response
Required
  • User ‚Üí Chat Router ‚Üí Orchestrator ‚Üí [search‚Üífilter‚Üírank‚Üíapply] ‚Üí Response

Fix

Add start_workflow tool; create WorkflowInvoker; connect ChatOrchestrator to StateGraph. Files: core/tools.py, routers/chat.py, agents/chat_orchestrator.py. Effort: 2 weeks

2 No Conversation Persistence RESOLVED

Chat history in-memory only. Users lose all conversations on reload. No Conversation/Message models or DB tables.

Fix

Create Conversation + Message models, Alembic migration, ConversationRepository, GET/POST conversation/message APIs, update frontend to load history. Effort: 1 week

3 LinkedIn Automation Incomplete

AgentTask/AgentState exist, Playwright installed; agent router uses asyncio.sleep(5) placeholder. No JobSearchAgent.execute() or ApplyAgent Easy Apply logic.

Fix

Implement JobSearchAgent.execute(), ApplyAgent.execute(), ResumeAgent.tailor(); wire agent router to real execution; add rate-limit handling. Effort: 3‚Äì4 weeks

4 Hardcoded Secrets in Git

.env with production secrets is git-tracked (JWT_SECRET_KEY, VAPI_API_KEY, NEW_RELIC_LICENSE_KEY, LANGCHAIN_API_KEY exposed).

Fix

Add .env to .gitignore; rotate all exposed secrets; use Docker secrets or secrets manager. Effort: 1 day

5 ApplicationStatus Enum Mismatch

Schema: DRAFT, APPLIED, UNDER_REVIEW, INTERVIEW_SCHEDULED, OFFER, REJECTED, WITHDRAWN. Tools use SUBMITTED, OFFER_RECEIVED etc. ‚Üí LLM generates invalid values, tool calls fail.

Fix

Update backend/app/core/tools.py to match schema enum. Effort: 2 hours
